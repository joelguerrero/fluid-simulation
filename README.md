## Usage

1. Download this repo and store it in your computer.
2. Open a terminal and go to the root directory of this folder.
3. Make sure you have installed the needed dependencies by typing:

```
$ pip install numpy
$ pip install matplotlib
```
4. Type to run:

```
$ python fluid.py
```
